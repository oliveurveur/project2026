Privacy Policy

Last updated: May 2026

1. Overview

This application is a private tool used solely by its developer to upload video content to their own TikTok account via the TikTok Content Posting API. It does not serve any public users and does not collect, process, or store personal data belonging to any third party.

2. Data Collected

The only data processed by this application is:

  • Video files created by the developer and stored locally on the developer's own machine.
  • TikTok OAuth access tokens, stored locally on the developer's machine and used solely to authenticate API requests.

No personal data from TikTok users or any other individuals is collected, stored, or processed.

3. Data Sharing

No data is shared with any third party. Access tokens are never transmitted outside of the direct communication between this tool and the TikTok API.

4. Data Retention

Access tokens are stored locally and can be deleted at any time by the developer. Video files remain on the developer's local machine and are not uploaded to any service other than TikTok.

5. TikTok API

This application uses the TikTok API in accordance with TikTok's API Terms of Service. The developer's use of TikTok data is limited to what is strictly necessary to publish video content to their own account.
